# Prisma

CRUD using Prisma with Vue.

## Installation

Using pnpm to install the front.

```bash
pnpm install
```

Using pnpm to install the front.

```bash
npm install
```
## Usage

To run Front End.

```bash
pnpm run dev
```

To run Back End.

```bash
npm run dev
```

To run Docker compose file.

```bash
docker compose up -d
```
